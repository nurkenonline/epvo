1. Назначение 
Автоматизация процесса сбора и обработки информации в ИС ЕПВО предназначена для 
формирования системы оценки и управления рисками (далее – СУР). Сведения по контингенту 
обучающихся, выпускников, профессорско-преподавательского состава и иные сведения 
вводятся в информационных системах всех ОВПО (далее – ИС ОВПО) и должны быть 
переданы в ИС ЕПВО посредством интеграционного взаимодействия.  
Обработка сведений производится в ИС ЕПВО, согласно требований. 

5 
 
2. Описание показателей субъективного критерия 
Все сведения по показателям, источником которых является ИС ОВПО, должны быть 
рассчитаны в разрезе ОВПО. 
Период сбора данных для типа критерия: 
 Профилактический контроль: 01.06.2025 – 25.11.2025* 
 Квалификационные требования: 01.01.2025 – 25.11.2025* 
*возможно будет установлен период, определенный Уполномоченным органом по 
предпринимательству 
 
2.1. 
Отсутствие обучающихся в ОВПО, участвующих в программах внешней и 
внутренней академической мобильности 
Тип: Профилактический контроль 
Из контингента обучающихся, переданных из ИС ОВПО, будут определены обучающиеся: 
 Статус - «Обучающийся», не находящийся в академическом отпуске; 
 Имеющие один из следующих приказов в статусе «Подтвержден»: 
 Приказ о выезде в рамках академической мобильности (выезд внутренний);  
 Приказ о выезде в рамках академической мобильности (выезд внешний); 
 Приказ о зачислении (о приезде в рамках внешней академической мобильности); 
 Приказ о зачислении (о приезде в рамках внутренней академической мобильности); 
 Период обучения пересекается с периодом сбора данных. 
В таблице students будут учтены обучающиеся, имеющие дополнительный статус «Выехавшие 
в рамках академической мобильности» students.academic_mobility, не находящиеся в 
академическом отпуске students.isinretire 
Передача сведений в ИС ЕПВО:  
Необходимо передавать в ЕПВО актуальный контингент обучающихся в таблицу students. В 
таблице students будут учтены обучающиеся со статусом - «Обучающийся» (isstudent=1), не 
находящиеся в академическом отпуске (students.isinretire)

2.3. 
Доля студентов первого года обучения, условно зачисленных в ОВПО, более 10% 
от общего контингента приема на текущий учебный год 
Тип: Профилактический контроль 

32 
 
Необходимо определить: 
 Общее количество приема обучающихся (бакалавриат, специалитет); 
 Количество условно зачисленных обучающихся. 
Из контингента обучающихся, переданных из ИС ОВПО, будут определены обучающиеся: 
 Статус - «Обучающийся», не находящийся в академ. отпуске; 
 Уровень образования – «бакалавриат» и «специалитет»; 
 Курс обучения – первый; 
 Дата приказа о зачислении на 1 курс – за текущий год, начиная с 01.06; 
Из выделенного контингента обучающихся будут определены условно-зачисленные 
обучающиеся, имеющие: 
 Статус - «Условно-зачисленный». 
Доля условно-зачисленных рассчитывается по формуле: 
Формула: А = (C*100%) / B 
A – Доля условно зачисленных 
B – Общий контингент приема обучающихся 
С – Количество условно зачисленных обучающихся 
По данному показателю процент условно зачисленных должен составлять менее 10%. 
Передача сведений в ИС ЕПВО:  
Данные будут определены из таблиц «Сведения об обучающихся» students и «Дополнительные 
сведения об обучающихся» student_info, описанных выше.  
Статус - «Условно-зачисленный» передается в поле conditionally_enrolled таблицы students

Тип: Профилактический контроль

Необходимо определить: 

33 
 
 Общее количество выпускников (бакалавриат, специалитет, интернатура) по каждому 
году. С 1991 по 2023, согласно данным КВПО. С 2024 года и позднее, согласно данным 
в ИС ЕПВО; 
 Количество выпускников, переданных в ИС ЕПВО по каждому году. 
Из контингента выпускников, переданных из ИС ОВПО, будут определены выпускники:

Доля оцифрованных сведений по выпускникам рассчитывается по формуле: 
Формула: Аn = (Cn*100%) / Bn 
A – доля оцифрованных сведений по выпускникам  
B – общее количество выпускников, согласно сведениям КВПО

n – год выпуска

2.23. Доля обучающихся в ОВПО на каждом годе обучения по уровням бакалавриата и

Тип: Квалификационные требования

(профильное направление): 
 Количество обучающихся, переведенных и восстановленных из зарубежных ОВПО; 
 Среднегодовой контингент обучающихся. 
 
Из контингента обучающихся, переданных из ИС ОВПО, будут определены обучающиеся:

 Система автоматически в каждом ОВПО подсчитывает количество обучающихся со 
статусом «Обучающийся», не находящихся в академическом отпуске, по месяцам 
текущего года: февраль, март, апрель, октябрь, ноябрь; 
 Полученные результаты сохраняются в системе; 
 Среднегодовой контингент обучающихся определяется как среднее арифметическое 
значений, рассчитанных на перечисленные даты.

каждому году обучения и академической степени рассчитывается по формуле: 
Аn = (Cn*100%) / Bn 
A – Доля переведенных и восстановленных обучающихся из зарубежных ОВПО по 
каждому году обучения заданной академической степени 
B – Среднегодовой контингент обучающихся каждого года обучения заданной 
академической степени 
С – Количество переведенных и восстановленных обучающихся граждан Казахстана из 
зарубежных ОВПО по каждому году обучения заданной академической степени 
n – год обучения (курс) 
По данному показателю процент переведенных из зарубежных ОВПО должен составлять не 
более 30% для соответствия. Далее будут подсчитаны все значения соответствия и не 
соответствия по всем годам. Если соответствий по годам больше или количество соответствий 
и несоответствий равно, итого по критерию будет зафиксировано соответствие.

2.24. Доля защит докторантов ОВПО менее 40% от общего количества выпускников в 
течении 2 лет после выпуска 
Тип: Квалификационные требования 
Будет определено: 
 Общее количество докторантов, окончивших обучение в {2023} году; 
 Количество защитившихся докторантов в течение 2 лет после окончания срока обучения. 
Для определения количества обучающихся докторантуры, которые закончили обучение 
в соответствующем году будет подсчитано общее количество: 
1. 
Обучающихся (students) со статусом «Отчислен» или «Выпускник» с академической 
степенью «Докторант по направлениям», «Докторант PhD» (degreeid in (6, 7)), для которых 
передан Приказ об отчислении с категорией «В связи с окончанием срока обучения 
докторантуры» с датой приказа за {2023} год; 
2. 
Выпускников (graduates) с академической степенью «Докторант по направлениям», 
«Докторант PhD» (degreeid in (6, 7)) с «Датой приказа о выпуске» (finishorderdate) за {2023} 
год, у которых отсутствует приказ об отчислении с указанной категорией.  
 Далее из общей выборки будут определены докторанты, которые защитились в течение 
2 лет после окончания срока обучения, то есть соответствующие одному из следующих 
условий: 
1. 
Передан данным ОВПО, как выпускник с академической степенью «Докторант по 
направлениям», «Докторант PhD» (graduates) и имеет признак «Докторант с защитой» 
(Докторанты, которые защитились в том же ОВПО, где учились). 
2. 
Передан другим ОВПО, как внешний слушатель (el_external_listener) с типом 
«Слушатель докторантуры PhD» / «Слушатель докторантуры по профилю» со статусом 
«Выпускник» и указанным основным ОВПО – ОВПО, для которого выполняется отчет 
(Докторанты, которые защитились другом ОВПО). 
 
Доля защитившихся докторантов от общего количества выпускников докторантуры 
определяется по формуле ниже: 
Формула: А = (C*100) / B, где 

62 
 
 
A – доля защитившихся докторантов, % 
 
B – общее количество выпускников докторантуры 
 
С – количество докторантов, защитившихся в течение 2 лет после выпуска. 
 
По данному показателю процент защитившихся докторантов должен составлять не менее 
40%.  
При отсутствии в ОВПО выпускников докторантуры, по критерию будет зафиксировано 
соответствие.
