## Категоризация по темам

### ОСМС (Страхование) — ~60 заявок
Большинство заявок связаны с тем, что статус студента/выпускника не отражается в базе ОСМС. Типовой ответ службы поддержки: необходимо проверить наличие приказа о зачислении с правильным `categoryid=104`, после чего данные переотправляются в ОСМС.

### Цифровые документы / ЕГОВ / Касмарт — ~30 заявок
Дипломы, студенческие билеты, справки не отражаются в eGov/Kaspi. Причины: некорректный ИИН, дата рождения, отсутствие привязки ГОП в таблице `graduates`.

### Реестр ОП / Учебные планы — ~40 заявок
Проблемы с включением, обновлением, изменением названий образовательных программ. Ключевой инсайт: для изменения уже одобренных ОП нужно обращаться в НЦРВО.

### Интеграция / API — ~15 заявок
Запросы инструкций, ошибки при передаче данных ППС, лабораторий, приказов. Swagger: `epvo.kz/isvuz/api/swagger-ui/index.html`.

### Финансирование (Стипендия) — ~15 заявок
Ошибки в отчетах о финансировании, отсутствие данных в карточках студентов.

### СУР / Нац. рейтинг — ~10 заявок
Вопросы по критериям оценки, показателям устойчивого развития, докторантам.

### Доступы / Пароли — ~15 заявок
Сброс паролей, создание пользователей, назначение ролей.

### Генерация дипломов — ~10 заявок
Ошибки ИИН, долгая загрузка, проблемы с ЭЦП.

### Разное (выпускники, eНбек, переводы) — ~10 заявок
