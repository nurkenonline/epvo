     Инструкция 
     для передачи сведений в модуль «Финансирование стипендии»
      
      Для автоматизации процессов формирования отчетов для выплаты стипендий обучающимся ОВПО в ЕПВО запускается модуль «Финансирование» (далее – модуль). Сведения по контиенгенту ОВПО, обучающихся по государственному обраовательному заказу для формирования отчетов в рамках модуля должны быть введены в информационных системах всех ОВПО и должны быть переданы в ЕПВО посредством интеграционного взаимодействия. В рамках данного модуля необходимо передавать следующие данные:
1. Передавать в ЕПВО сведения по контингенту обучающихся/слушателей подготовительного отделения (в случае наличия) ОВПО, заполнив соответствующие таблицы, которые обозначены в приложении к письму № 4-01-07-02-04 /4905-И от 29.08.2023;
2. Передавать в ЕПВО сведения по контингенту слушателей подготовительного отделения (в случае наличия) ОВПО, заполнив таблицы students, student_info, foundation_group;
3. Передавать в ЕПВО сведения по льготным категориям обучающихся/слушателей подготовительного отделения, а также по квотам обучающихся, заполнив таблицу studentbenefits;
4. Передавать в ЕПВО сведения по приказам обучающихся/слушателей подготовительного отделения, заполнив соответствующие таблицы, которые обозначены в приложении к письму № 4-01-07-02-04 /4905-И от 29.08.2023;
Список приказов, требуемых для формирования отчетов:
• Приказ о зачислении
• Приказ об отчислении 
• Подготовительное отделение. Приказ о зачислении
• Подготовительное отделение. Приказ об отчислении
• Приказ о смене формы оплаты
• Приказ об академическом отпуске
• Приказ о продлении академического отпуска
• Приказ о выходе из академического отпуска
• Приказ о переводе (с одной ГОП на другую)
• О выезде в рамках академической мобильности
• О выезде на обучение в рамках программы обмена «Двудипломное образование»
• Приказ о возвращении из командировки
• О завершении обучения в рамках программы обмена «Двудипломное образование»
• Приказ о назначении стипендии
• Подготовительное отделение. Приказ о назначении стипендии
• Приказ о прекращении выплаты стипендии
5. Передавать в ЕПВО сведения по успеваемости обучающихся (таблицы transcript, generalexamsmarks);
6. Передавать в ЕПВО сведения по назначенным стипендиям обучающимся/слушателей подготовительного отделения, заполнив соответствующие таблицы, которые обозначены в приложении к письму№ 4-01-07-02-04 /4905-И от 29.08.2023;
      
      Для отображения всех необходимых данных по обучающимся в отчетах для выплаты стипендий в модуле «Финансирование» в ЕПВО, высылаем перечень дополнительных таблиц и полей для передачи через API-сервис:
• student_info (iic, bic, winter_admission)
• studentbenefits (full_provision)
• sectionpersons (movement_date, overall_performance)
• scholarship (scholarship_award_year, scholarship_award_term, overall_performance)
• transcript (not_included_scholarship)
• foundation_group
• ordersadditional
• retires
Формат заполнения полей и ссылки на таблицы смотреть в http://epvo.kz/isvuz/api.

overall_performance
Да
Числовой
Общая успеваемость за указанный учебный год и академический период, на основе которой назначена стипендия в рамках заданного приказа о назначении стипендии
Возможные значения:
• 5 - если у обучающегося все оценки в выбранном академическом периоде – отличные. 
• 4 - если есть хотя бы одна оценка «Хорошо» и нет удовлетворительных/неудовлетворительных оценок. 
• 3 - если есть хотя бы одна удовлетворительная оценка и нет неудовлетворительных оценок. 
• 2 – если есть хотя бы одна неудовлетворительная оценка
• 1 – если стипендия назначается без учета успеваемости обучающегося (например, для 1 курса 1 академ. периода, либо для академических степеней докторантура (по направлениям, научно-педагогическое) /резидентура, подготовительное отделение)
• 0 - если нет записей в транскрипте обучающегося за выбранный учебный период, по которому назначается стипендия
