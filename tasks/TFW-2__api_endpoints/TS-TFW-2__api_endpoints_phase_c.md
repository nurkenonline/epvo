## 1. Цель
Задокументировать специализированные API-эндпоинты ЕПВО: методы для получения списка зачисленных студентов (Enrollment, Приёмные кампании) и методы для работы с файлами.

## 2. Scope

### In Scope
- **Enrollment API:** методы для получения списков зачисления:
  - бакалавриат (`/grant-enrollment/findAll` или аналогичный, если опираться на спецификацию).
  - магистратура (`/magistracy-enrollment/findAll`).
- **File API:** методы получения и управления файлами (например, `/v1/file/{fileId}`).
- Структура запросов, параметры спецификации и ожидаемые форматы ответов (на базе OpenAPI или стандартных предположений).

### Out of Scope
- Практическое скачивание файлов через API (мы лишь описываем эндпоинт).
- Модули базового контингента и справочников (рассмотрены в Phase A и Phase B).
